Initial Setup and Configuration of an IBM PowerPC server

If you want to administrate IBM PPC servers via a virtualized HMC then some initial server configuration needs to be completed first. The most important part of this is the initial configuring of the server's Advanced System Management Interface which manages networking and credentials for the machine. This networking and credentials configuration is important for adding machines to the HMC which is required if we are to use the HMC at all. The carousel slider below is an instruction set for the needed initial server and ASMI configuration. If you encounter trouble in your setup procedure see the Help page for additional IBM and 3rd party resources for troubleshooting your server setup.
