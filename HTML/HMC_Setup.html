Installing the HMC and Using It To Administrate PowerPC Servers

The IBM HMC can easily be installed to a VM. The instruction set below assumes that you already have VMware Player installed. If this is true, install the HMC .iso to a VM like you would any other operating system. Then, open up the virtual network editor and set it to bridge VMware's networking via whatever ethernet network interface the HMC VM will be using. The carousel slider below is an instruction set for installing an HMC and using it to administrate PPC servers. If you encounter trouble in your setup procedure see the Help page for additional IBM and 3rd party resources for troubleshooting your server setup.
